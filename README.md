Polymorphic Banking System (Practice Project)



Small C++ project created as practice for the course.

It focuses on OOP, inheritance, and dynamic polymorphism by modeling different types of bank accounts.



Features



Abstract base Account class



Derived account types (e.g., Savings, Checking)



Polymorphic deposit, withdraw, and print functions



Uses base-class pointers and vectors



Simple utilities for displaying and operating on accounts



Purpose



This project was built only for learning and practicing C++ OOP and polymorphism.
